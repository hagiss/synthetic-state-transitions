## GitHub Pages

Publish this repository from the repository root using GitHub Pages:

1. Push these files to `https://github.com/hagiss/synthetic-state-transitions`.
2. Open repository Settings -> Pages.
3. Set the source to the main branch and the root folder.
4. The page should be available at `https://hagiss.github.io/synthetic-state-transitions/`.

Update `index.html` with the final author list, paper link, and BibTeX metadata when the camera-ready details are final.
